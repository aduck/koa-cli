## 开始

请先配置config文件 dev.js开发配置 prod.js生产配置

```
npm install
npm run dev # 开发
npm run build # 生产
```

### 目录

```
api     三方接口
config  配置
control 业务逻辑
job     任务processor
log     日志
middle  中间件
model   封装数据库操作
router  路由
utils   通用（service）
```
